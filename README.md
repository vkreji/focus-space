# focus space

a clean, practical productivity tool for managing tasks and staying focused.

> **Note:** All data is stored locally in your browser. Nothing is sent to any server.

## what it does

**focus space** is a personal workspace that combines:
- **pomodoro timer** - 25/45/60 minute focus sessions
- **task management** - organize your work with priorities
- **daily focus** - set intentions for the day
- **progress tracking** - see your daily stats

## features

### timer
- preset durations: 25, 45, or 60 minutes
- start/pause/reset controls
- automatic session completion tracking
- browser notifications when sessions end

### tasks
- create tasks with titles and notes
- set priority levels (high/normal/low)
- mark tasks as complete
- double-click to edit
- persistent storage

### daily focus
- quick checklist for daily intentions
- check off items as you complete them
- resets each day

### stats
- track minutes focused today
- count completed sessions
- daily progress visualization

## how to use

1. open `index.html` in your browser
2. set a timer for your work session
3. add tasks you want to complete
4. set your daily focus items
5. start working!

### keyboard shortcuts
- `Enter` in task/focus inputs to save
- `Escape` to close modals
- double-click tasks to edit

## technical

- pure HTML/CSS/JavaScript
- localStorage for persistence
- browser notifications API
- responsive design
- no dependencies
